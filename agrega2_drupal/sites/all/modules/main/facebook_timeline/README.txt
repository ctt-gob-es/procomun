This module has been implemented for Drupal 7. It includes the facebook timeline
widget inside a block.

Installation / Uninstallation

As usual in Drupal 7. This URL should provide more information about the process
https://drupal.org/documentation/install/modules-themes/modules-7

Configuration

Administrators must provide the values for the fanpage name. It should be
configured as usual in the admin/structure/block section.
